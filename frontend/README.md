# Scipioni Club Frontend

Questa cartella e' riservata all'app mobile Flutter per iOS e Android.

## Stato

Il progetto Flutter non e' stato bootstrapato in questo ambiente perche' `flutter` non e' installato.

## Bootstrap previsto

Quando l'SDK sara' disponibile, il comando previsto e':

```bash
flutter create .
```

Successivamente:

```bash
flutter pub get
flutter run
```

## Struttura target

- `lib/app/`
- `lib/features/auth/`
- `lib/features/home/`
- `lib/features/events/`
- `lib/features/bookings/`
- `lib/features/profile/`
- `lib/shared/`
